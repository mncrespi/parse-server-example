# Parse Server with Dashboard

A parse Server Demo, App Backend Push Notification.

## Installation

```sh
git clone https://github.com/mncrespi/parse-server-example.git parse-server
cd parse-server
npm i
```


## Run

```sh
npm start
```

## Parse-Server

> URL: http://localhost:8080/parse/
